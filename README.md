Overview

This is the personal portfolio website of Abayomi Odusanya, a Frontend Web Developer. The website is designed to showcase my professional skills, projects, and contact information in a modern, responsive, and user-friendly interface.

It highlights:

My web development projects

Technical skills

Contact information with integrated social media links

The portfolio is built to be responsive, interactive, and compatible with light/dark mode, making it suitable for both desktop and mobile viewing.

Features

Responsive Design: Fully optimized for mobile, tablet, and desktop.

Dark/Light Mode Toggle: Allows users to switch between dark and light themes.

Project Showcase: Interactive project cards with live demo links and source code.

Skills Section: Visual representation of technical competencies.

Contact Form: Fully styled form for user messages and professional contact details.

Smooth Animations: Fade-in effects on scroll and hover effects for interactive elements.

Social Integration: Direct links to GitHub, LinkedIn, and X (Twitter).

Technologies Used

HTML5 – Structure and semantic markup

CSS3 – Styling, flexbox, grid, transitions, and animations

JavaScript (Vanilla) – Dark/light mode toggle, scroll animations

Font Awesome – Icons for social links and UI elements
